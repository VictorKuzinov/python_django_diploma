# 📦 Django Diploma: Интернет-магазин Megano

Проект интернет-магазина на Django в рамках дипломной работы.

## 🔧 Текущий статус (Checkpoint 1)

-   Django-проект создан
-   Настройки вынесены в `.env`
-   Подключён frontend-пакет `diploma-frontend`
-   Проект воспроизводим через `requirements.txt`
-   Сервер успешно запускается

------------------------------------------------------------------------

# 🚀 Запуск проекта

## 1. Клонировать репозиторий

``` bash
git clone <repo_url>
cd python_django_diploma
```

------------------------------------------------------------------------

## 2. Создать виртуальное окружение

``` bash
python -m venv .venv
source .venv/bin/activate  # Linux / WSL
```

Windows:

``` bash
.venv\Scripts\activate
```

------------------------------------------------------------------------

## 3. Установить зависимости

``` bash
pip install -r requirements.txt
```

------------------------------------------------------------------------

## 4. Создать файл окружения

Скопировать шаблон:

``` bash
cp .env.template .env
```

Минимальное содержимое `.env`:

    DJANGO_SECRET_KEY=your_secret_key
    DJANGO_DEBUG=True
    DJANGO_ALLOWED_HOSTS=127.0.0.1,localhost

    DJANGO_DB_ENGINE=sqlite
    DJANGO_DB_NAME=db.sqlite3

------------------------------------------------------------------------

## 5. Применить миграции

``` bash
python manage.py migrate
```

------------------------------------------------------------------------

## 6. Запустить сервер

``` bash
python manage.py runserver
```

Открыть в браузере:

    http://127.0.0.1:8000/

Должна открыться стартовая страница магазина Megano.

------------------------------------------------------------------------

# 🏗 Архитектура проекта

-   `config/` --- основной Django-проект
-   `diploma-frontend/` --- подключаемый frontend-пакет
-   `requirements.txt` --- зависимости проекта
-   `.env` --- переменные окружения (не хранится в репозитории)
-   `.env.template` --- шаблон переменных

Frontend устанавливается в editable-режиме:

    -e ./diploma-frontend
